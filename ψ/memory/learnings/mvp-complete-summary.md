# 🦀 MVP ฟาร์มปูทองหลาง - Complete Product Summary

**สร้างเมื่อ:** 2026-03-02  
**เจ้าของ:** แบงค์ (Bank)  
**จัดการโดย:** กุ้ง Oracle (Kung)  
**ระบบ:** Oracle + RAS + Individual Boxes  

---

## 🎯 ภาพรวม MVP (Executive Summary)

```
Concept: ฟาร์มปูทองหลาง indoor แบบ RAS + Individual Boxes
Scale: MVP เริ่มต้น 20 ตัว → scale เป็น 300-500 ตัว
Time: Setup 7 วัน + เลี้ยง 4-5 เดือน (hard-shell) หรือ 3-4 สัปดาห์ (soft-shell)
Budget: 15,000-19,500 บาท (setup) + 800 บาท/เดือน (operating)
ROI: ประหยัดแรงงาน + เพิ่ม survival rate + คุณภาพปูดีขึ้น
```

## 📋 Phase 0: ตัดสินใจ (ก่อนเริ่ม)

### ✅ Decision 1: เลือก Farming Path

```
□ Path A: Full Cycle (ซับซ้อนสุด)
  - Crablet → Grow-out (+substrate) → Soft-shell → Harvest
  - เวลา: 4-5 เดือน
  - งบ: สูงสุด
  - เรียนรู้: เยอะสุด

□ Path B: Soft-shell Only (แนะนำสำหรับ MVP)
  - ซื้อปู 50-150g → Soft-shell unit (no substrate) → Harvest
  - เวลา: 3-4 สัปดาห์
  - งบ: ปานกลาง
  - เรียนรู้: เร็ว, focused
  - ข้อดี: ไม่ต้องใช้ทราย, cycle สั้น, ราคาขาย 4-5x

□ Path C: Hard-shell Only (ง่ายสุด)
  - Crablet → Grow-out (+substrate) → Harvest hard-shell
  - เวลา: 4-5 เดือน
  - งบ: ต่ำสุด
  - เรียนรู้: พื้นฐาน

⭐ แนะนำ: Path B (Soft-shell Only)
```

### ✅ Decision 2: เลือก Budget Option

```
□ Option A: ประหยัด (15,000 บาท)
  - ไม่มี UV Sterilizer / ไม่มี DO meter
  - มี: Protein Skimmer + K1 Moving Bed + Backup systems
  - เหมาะ: ทดลองแต่ยังมีระบบสำรอง

□ Option B: สบายใจ (17,000 บาท) ⭐ แนะนำ
  - มี UV Sterilizer + Protein Skimmer + K1 + Backup
  - ไม่มี DO meter
  - Balance: ราคา + ความสบายใจ + ระบบแข็งแรง

□ Option C: ครบครัน (19,500 บาท)
  - มี UV + DO meter + Skimmer + K1 + Backup
  - สบายใจสุด ครบทุกอย่าง
```

### ✅ Decision 3: สถานที่พร้อม?

```
□ พื้นที่: 2×3 เมตร (6 ตรม.)
□ หลังคาครอบ (ไม่โดนฝน/แดด)
□ ไฟฟ้า 220V
```

## 🔧 Phase 2: Setup Steps (7 วัน)

### Day 1: Setup Core System

```
เวลา: 2-3 ชม.

ขั้นตอน:
  1. □ วาง Tank 1000L บนพื้นเรียบ
  2. □ ตั้ง Pump ใกล้ Tank
  3. □ ต่อท่อ suction (Tank → Pump)
  4. □ ติดตั้ง Sponge Filter หลัง Pump
  5. □ ติดตั้ง Bio Filter (ใส่ bio-balls)
  6. □ ติดตั้ง Air Pump + Air Stones
  7. □ ทดสอบ Pump + Air (ใช้น้ำจืด 200L)

Checklist:
  □ Pump ทำงานปกติ?
  □ Air bubbles ออกมา?
  □ ไม่มีท่อรั่ว?
```

### Day 2: ต่อท่อ Supply Side (น้ำเข้า)

```
เวลา: 3-4 ชม.

ขั้นตอน:
  1. □ ต่อท่อจาก Bio Filter → Distribution Manifold
  2. □ สร้าง Distribution Manifold (ท่อแนวนอน 4 เส้น)
  3. □ ใส่ Ball Valve แต่ละแถว
  4. □ เจาะรูกล่อง 20 ใบ (inlet + outlet)
  5. □ ต่อ Hose ¼" จาก Manifold → กล่อง (20 เส้น)

Checklist:
  □ ทุกกล่องมีท่อน้ำเข้า?
  □ Valves หมุนได้?
  □ กาว PVC แห้งแล้ว?
```

### Day 3: ต่อท่อ Return Side (น้ำออก)

```
เวลา: 2-3 ชม.

ขั้นตอน:
  1. □ ต่อ Hose ¼" จากกล่อง → Collection Manifold
  2. □ รวมท่อทั้งหมด → Main Return Pipe
  3. □ ต่อกลับไป Tank
  4. □ ทดสอบระบบ (เปิด Pump ดูน้ำไหล)

Checklist:
  □ น้ำไหลจากกล่องกลับไป Tank?
  □ ทุกกล่องน้ำไหลเท่ากัน?
  □ ไม่มีรั่ว?
```

### Day 4: เตรียมกล่อง + ทราย (ถ้า Path A/C)

```
เวลา: 2-3 ชม.

ขั้นตอน:
  1. □ ล้างทราย (ล้างจนน้ำใส)
  2. □ ใส่ทรายในกล่อง 3-5 cm (20 กล่อง)
  3. □ เรียงกล่องเป็นระเบียบ
  4. □ ตรวจสอบ inlet/outlet ไม่ถูกทรายทับ

ถ้า Path B (Soft-shell):
  □ ข้ามขั้นตอนนี้ (ไม่ต้องใช้ทราย)
```

### Day 5: เตรียมน้ำ + เริ่ม Cycling

```
เวลา: 2 ชม. + เริ่มนับ Cycling 2-4 สัปดาห์

ขั้นตอน:
  1. □ เติมน้ำจนเต็ม Tank (1000L)
  2. □ เติม Sea Salt Mix 15-20 kg
  3. □ เปิด Pump + Air + Protein Skimmer ให้วิ่ง
  4. □ รอ 24 ชม. ให้เกลือละลาย
  5. □ วัดค่าน้ำ:
     - Salinity: 15-20‰ ✓
     - pH: 7.5-8.5 ✓
     - Temp: 28-30°C ✓
  6. □ เติม Probiotics (Bacteria Starter)
  7. □ เติม Ammonia Source: อาหารปลา 1 หยิบมือ ทิ้งให้เน่า
  8. □ เริ่ม Cycling!

Checklist:
  □ เกลือละลายหมดแล้ว?
  □ ค่าน้ำถูกต้อง?
  □ Protein Skimmer มีฟองบ้างไหม?
```

### Day 6-7: ทดสอบระบบ + Cycling ต่อเนื่อง

```
ขั้นตอน:
  1. □ รันระบบ 24 ชม. ต่อเนื่อง
  2. □ Monitor ทุก 4 ชม.:
     - Pump ทำงาน?
     - Air ทำงาน?
     - น้ำไหลปกติ? (ทุกกล่องน้ำไหลผ่าน)
     - ไม่รั่ว?
  3. □ วัดค่าน้ำ + Ammonia + Nitrite ทุกวัน
  4. □ บันทึกข้อมูลใน psi/memory/logs/

Checklist:
  □ ระบบ stable 24 ชม.?
  □ ค่าน้ำ stable?
```

### สัปดาห์ 2-4: Cycling ระบบ (ห้ามข้าม!)

```
⚠️ สำคัญมาก: ห้ามใส่ปูก่อน cycle เสร็จ!

ทุกวัน:
  □ วัด Ammonia (NH3)
  □ วัด Nitrite (NO2)
  □ บันทึกค่าใน log

จะเห็น pattern นี้:
  สัปดาห์ 1: Ammonia พุ่งขึ้น ↑↑ (ปกติ)
  สัปดาห์ 2: Ammonia ลด ↓ + Nitrite พุ่ง ↑↑
  สัปดาห์ 3-4: ทั้งคู่ลดลง ↓↓

✅ พร้อมใส่ปูเมื่อ: Ammonia = 0 และ Nitrite = 0

วิธีเร่ง cycle:
  - ใช้ filter media จากตู้ปลาเก่า
  - เปิดปั๊มลมแรงๆ
  - รักษาอุณหภูมิ 28-30°C
```

## 🚨 Emergency Protocols

### Pump Failure

```
1. □ หยุดให้อาหารทันที
2. □ เปิด Air Pump แรงขึ้น
3. □ แจ้งแบงค์ทันที
4. □ ซ่อม/เปลี่ยน Pump เร็วที่สุด
5. □ บันทึกใน log
```

### Water Quality Crisis

```
1. □ วัดค่าน้ำทุกตัว (pH, Salinity, Temp, DO)
2. □ เปลี่ยนน้ำ 50% (ถ้าจำเป็น)
3. □ แยกปูที่แข็งแรง
4. □ แจ้งแบงค์
5. □ บันทึกใน log
```

### Mass Mortality (>20% หรือ 4 ตัวขึ้นไป)

```
1. □ หยุดให้อาหาร
2. □ ตรวจสอบทุก parameter
3. □ ถ่ายรูปปูที่ตาย
4. □ แจ้งแบงค์ทันที
5. □ เก็บตัวอย่างน้ำ
6. □ บันทึกใน log
```
