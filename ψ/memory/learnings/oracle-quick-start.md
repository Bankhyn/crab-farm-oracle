## 🔧 Monthly Maintenance (ทุกเดือน - 1 ชม.)

### สัปดาห์ที่ 2 ของเดือน

```
1. □ ล้าง Sponge Filter
   - ถอดออก
   - ล้างด้วยน้ำในระบบ (ห้ามใช้น้ำประปา)
   - ใส่กลับ

2. □ เปลี่ยนทราย (ถ้า Path A/C)
   - ทุก 2-4 สัปดาห์
   - หรือถ้าสกปรกมาก

3. □ วัดน้ำหนัก sample (5-10 ตัว)
   - จับจากกล่องสุ่ม
   - ชั่งน้ำหนัก
   - บันทึก
   - ใส่กลับ

4. □ ถ่ายรูปเปรียบเทียบ
   - ถ่ายรูปปู
   - เทียบกับรูปเดือนก่อน
   - เห็น growth ชัดเจน

เวลา: 1 ชม.
```

## 🚨 Emergency Checklist (ทำทันทีถ้ามีปัญหา)

### ปูตาย > 2 ตัว/วัน

```
1. □ หยุดให้อาหาร
2. □ วัดค่าน้ำทันที (Temp, Salinity, pH)
3. □ ตรวจสอบ Pump + Air
4. □ ถ่ายรูปปูที่ตาย
5. □ แจ้งกุ้ง → กุ้งจะแจ้งแบงค์
6. □ บันทึกใน log
```

### Pump พัง

```
1. □ ปิดสวิตช์ Pump
2. □ เปิด Air Pump แรงขึ้น
3. □ หยุดให้อาหาร
4. □ แจ้งกุ้ง → กุ้งจะแจ้งแบงค์
5. □ ซ่อม/เปลี่ยน Pump
```

### ไฟดับ

```
1. □ ใช้ Air Pump สำรอง (ถ้ามี)
2. □ หยุดให้อาหาร
3. □ รอไฟฟ้ากลับมา
4. □ ถ้าไฟดับ > 2 ชม. → แจ้งกุ้ง
```

## 💡 Tips จากกุ้ง

### การให้อาหาร

```
เดือนที่ 1-2:
  - 2 มื้อ/วัน (เช้า + เย็น)
  - 5-8% น้ำหนักตัวรวม/วัน
  - ตัวอย่าง: ปู 20 ตัว × 50g = 1000g
    → ให้ 50-80 g/วัน (แบ่ง 25-40 g/มื้อ)

เดือนที่ 3+:
  - 1 มื้อ/วัน (เช้า)
  - 3-5% น้ำหนักตัวรวม/วัน

สังเกต:
  - ถ้าปูกินหมดเร็ว → เพิ่ม
  - ถ้าเหลือ → ลด
  - ถ้าไม่กิน → หยุด 1 มื้อ + ตรวจสอบ
```

### การวัดค่าน้ำ

```
เวลาวัด: เช้า (8:00-9:00) ทุกวัน

Temperature:
  - ใช้ Thermometer digital
  - จุ้มในน้ำ 30 วินาที
  - อ่านค่า

Salinity:
  - ใช้ Refractometer
  - หยดน้ำ 1 หยดบนกระจก
  - มองผ่านช่องมอง
  - อ่านค่า (15-20‰)

pH:
  - ใช้ pH meter
  - จุ้มในน้ำ 30 วินาที
  - อ่านค่า (7.5-8.5)
  - ล้างด้วยน้ำกลั่นหลังใช้
```

### การสังเกตปู

```
สุขภาพดี:
  ✅ ขยับแข็งแรง
  ✅ กินอาหาร
  ✅ สีสด
  ✅ ไม่มีบาดแผล

สุขภาพแย่:
  ❌ นอนนิ่งๆ
  ❌ ไม่กินอาหาร
  ❌ สีซีด
  ❌ มีจุดดำ/แผล

ลอกคราบ (Molting):
  - ปูจะซ่อนตัว
  - ไม่กินอาหาร 1-2 วัน
  - หลังลอกจะตัวใหญ่ขึ้น
  - ประมาณ 3-7 วันตัวแข็ง
```
