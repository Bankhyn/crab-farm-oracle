## 10. Project Lifecycle

```
Seed --> Grow --> Graduate
```

- **Seed**: Idea captured, initial research
- **Grow**: Active development, experiments
- **Graduate**: Mature enough for its own repo/system

For Kung: Each harvest cycle can follow this pattern.
Each IoT experiment can follow this pattern.
