## 1. RAS คืออะไร?

### ความหมาย

```
RAS = Recirculating Aquaculture System
     = ระบบเพาะเลี้ยงสัตว์น้ำแบบหมุนเวียนน้ำ

หลักการ:
  ใช้น้ำซ้ำ ไม่ต้องเปลี่ยนบ่อย
  กรองของเสีย + เชื้อโรค
  ควบคุมคุณภาพน้ำ
  ประหยัดน้ำ + ควบคุมสภาพแวดล้อมได้
```

### เปรียบเทียบ

```
แบบดั้งเดิม (บ่อดิน):
  ❌ ต้องเปลี่ยนน้ำบ่อย
  ❌ คุมคุณภาพน้ำยาก
  ❌ โรคระบาดง่าย
  ❌ ใช้พื้นที่เยอะ

แบบ RAS (ปิด/ในร่ม):
  ✅ ใช้น้ำน้อย (เติมแค่ 10-20% ต่อสัปดาห์)
  ✅ คุมคุณภาพน้ำได้แม่น
  ✅ โรคระบาดยาก
  ✅ ใช้พื้นที่น้อย
```

## 2. Components ทั้งหมดมีอะไรบ้าง

### แบบแบ่งตามหน้าที่

```
A. แท็งก์น้ำ (Water Storage)
   └─ Central Tank 1000L

B. ระบบปั๊ม (Pumping System)
   ├─ Water Pump (½ HP)
   └─ ท่อ PVC ต่างๆ

C. ระบบกรอง (Filtration System)
   ├─ Mechanical Filter (Sponge)
   ├─ Biological Filter (Bio-balls/Media)
   └─ UV Sterilizer (optional)

D. ระบบให้ออกซิเจน (Aeration System)
   ├─ Air Pump
   ├─ Air Tubing
   └─ Air Stones

E. กล่องเลี้ยงปู (Crab Boxes)
   ├─ 20 กล่อง (30×40×15 cm)
   ├─ ทราย (Substrate)
   └─ Inlet/Outlet pipes

F. ระบบท่อ (Piping System)
   ├─ Main pipes (ท่อหลัก)
   ├─ Distribution pipes (ท่อแยก)
   ├─ Valves (วาล์วควบคุม)
   └─ Fittings (ข้อต่อต่างๆ)

G. เครื่องมือวัด (Monitoring Tools)
   ├─ pH Meter
   ├─ Refractometer (วัดความเค็ม)
   ├─ Thermometer
   └─ DO Meter (optional)
```

### Flow Rate

```
ปั๊ม ½ HP = 500-1000 ลิตร/ชั่วโมง
Tank 1000L = น้ำหมุนเวียน 1 รอบทุก 1-2 ชม.

กฎทองคำ:
  ✅ น้ำควรหมุนเวียน 6-12 รอบ/วัน
  ✅ สำหรับ 1000L → 6000-12000 ลิตร/วัน
  ✅ ปั๊ม ½ HP (500 L/hr) = 12000 L/24hr = OK!
```

## 4. รายละเอียดแต่ละส่วน

### A. Central Tank (แท็งก์น้ำหลัก)

```
หน้าที่:
  ✅ เก็บน้ำหลัก (1000L)
  ✅ ผสมเกลือทะเล
  ✅ ให้ที่อยู่ของแบคทีเรียบางส่วน

ขนาด: 1000L (1 คิว)
วัสดุ: 
  - พลาสติก (PE/PP) ← แนะนำ
  - หรือ ใยสาคูเคลือบ epoxy

ราคา: 1,500-3,000 บาท

รูปร่าง:
  - วงกลม (ดีสุด - น้ำไหลดี)
  - สี่เหลี่ยม (OK)

อุปกรณ์ใน Tank:
  - Air stone (ให้ออกซิเจน)
  - Heater (ถ้าจำเป็น - 28-30°C)
  - Thermometer (วัดอุณหภูมิ)

การเตรียมน้ำใน Tank:
  1. เติมน้ำประปา 1000L
  2. เติม sea salt mix: 15-20 kg
  3. รอ 24 ชม. ให้เกลือละลาย
  4. วัด salinity: 15-20‰ (ppt)
  5. วัด pH: 7.5-8.5
  6. เติม probiotics (แบคทีเรียดี)
```

---

### B. Pump (ปั๊มน้ำ)

```
หน้าที่:
  ✅ ดึงน้ำจาก tank
  ✅ ส่งน้ำผ่าน filter
  ✅ กระจายน้ำไปกล่อง

ขนาดที่แนะนำ: ½ HP (350-400W)

Spec:
  - Flow rate: 500-1000 L/hr
  - Head height: 2-3 เมตร
  - แรงดัน: เพียงพอส่งน้ำขึ้นไป

ราคา: 800-1,500 บาท

ยี่ห้อที่แนะนำ:
  - Hitachi
  - Mitsubishi
  - หรือ brand ท้องถิ่น

การติดตั้ง:
  1. วาง pump ใกล้ tank (ด้านล่าง)
  2. ต่อท่อ suction (ดูดน้ำจาก tank)
  3. ต่อท่อ discharge (ส่งน้ำไป filter)
  4. ติด strainer (กรองเศษใหญ่) ที่ท่อ suction
```

### C. Mechanical Filter (กรองของเสียใหญ่)

```
หน้าที่:
  ✅ กรองเศษอาหาร
  ✅ กรองขี้ปู
  ✅ กรองสิ่งสกปรกใหญ่

ประเภท:

Option 1: Sponge Filter (ง่ายสุด)
  - วัสดุ: ฟองน้ำ (sponge)
  - ขนาด: 30×30×5 cm
  - ราคา: 100-300 บาท
  - ทำความสะอาด: ล้างทุก 1-2 สัปดาห์

Option 2: Filter Sock (ถุงกรอง)
  - วัสดุ: ผ้ากรอง 200 micron
  - ขนาด: 4 นิ้ว
  - ราคา: 50-150 บาท/ใบ
  - เปลี่ยนทุก 3-7 วัน

Option 3: Drum Filter (แพง)
  - อัตโนมัติ
  - ราคา: 10,000+ บาท
  - ไม่จำเป็นสำหรับ MVP

วิธีติดตั้ง:
  1. วางหลัง pump
  2. ก่อน biological filter
  3. ต่อท่อเข้า-ออก
```

### D. Protein Skimmer (ดึงสารอินทรีย์ละลายน้ำ)

```
หน้าที่:
  ✅ ดึงสารอินทรีย์ละลายน้ำ (DOC) ออก
  ✅ ลดภาระ bio-filter
  ✅ ทำน้ำใส ลดกลิ่น
  ✅ จำเป็นสำหรับระบบน้ำเค็ม/กร่อย

หลักการ:
  ฟองอากาศจับสารอินทรีย์ → ดันขึ้นเป็นฟอง → เก็บใน cup
  (ทำงานได้ดีในน้ำเค็ม เพราะฟองเสถียรกว่าน้ำจืด)

ขนาด: สำหรับ 1000L → ตัวเล็กพอ

ตัวเลือก:
  ⭐ Bubble Magus QQ1 / QQ2 (แนะนำ)
    - ราคา: 1,200-2,000 บาท
    - ใช้กับตู้/ถัง 100-300L (ใช้ 2-3 ตัว)

  ยี่ห้อจีน (ประหยัด)
    - ราคา: 800-1,500 บาท
    - ค้น Shopee: "protein skimmer aquarium"
    - เลือกขนาดสำหรับ 500-1000L

วิธีติดตั้ง:
  1. ต่อหลัง mechanical filter
  2. ก่อน biological filter
  3. แขวน/วางข้าง tank หรือ sump
  4. ปรับระดับน้ำใน skimmer ให้ฟองพอดี

บำรุงรักษา:
  - ล้าง collection cup ทุก 2-3 วัน
  - ล้าง neck ทุกสัปดาห์ (ฟองจะเกาะ)

⚠️ สำคัญ:
  - ต้องใช้กับน้ำเค็ม/กร่อยเท่านั้น (น้ำจืดฟองไม่เสถียร)
  - ช่วง break-in 1-2 สัปดาห์ ฟองอาจยังไม่ดี → อดทน
```

### E. Biological Filter — K1 Moving Bed (กรองแบคทีเรีย)

```
หน้าที่:
  ✅ ให้ที่อยู่แบคทีเรียGood
  ✅ ย่อยสลายแอมโมเนีย (NH3) → ไนเตรท (NO3)
  ✅ ทำให้น้ำปลอดภัย

ขนาด: 50-100 ลิตร (5-10% ของ Tank)

วัสดุกรอง — ⭐ K1 Media Moving Bed (แนะนำ):
  - รูปร่าง: เม็ดพลาสติกเล็กๆ ลอยน้ำได้
  - หลักการ: ปั๊มลมตีให้ K1 หมุนวน → พื้นที่ผิวสัมผัสน้ำมากกว่า bio-balls
  - ปริมาณ: 3-5 ลิตร ต่อ 1000L น้ำ (น้อยกว่า bio-balls แต่ประสิทธิภาพดีกว่า)
  - ราคา: 200-400 บาท (3-5 ลิตร)
  - ค้น: "K1 media", "มีเดียลอย", "MBBR media"

  ทำไมดีกว่า Bio-balls:
    - พื้นที่ผิว: K1 = 500-800 m²/m³ vs Bio-balls = 200-300 m²/m³
    - Self-cleaning: media หมุนไม่อุดตัน
    - ประหยัดที่: ใช้ 3-5L แทน 50L bio-balls
    - เท่ากัน/ถูกกว่า ราคาต่อประสิทธิภาพ

ทางเลือก (ถ้าหา K1 ไม่ได้):
  Option 2: Bio-balls 50L → ราคา 300-600 บาท
  Option 3: Ceramic rings 50L → ราคา 500-1,000 บาท

Filter Housing (ภาชนะใส่ media):
  - ใช้ถังพลาสติกทึบ 20-50L (ถังน้ำ PE ราคาถูก)
  - เจาะรู inlet ด้านล่าง + outlet ด้านบน (ให้น้ำไหลจากล่างขึ้นบน)
  - ใส่ตะแกรงกั้น (กัน media ไหลออก) ที่ outlet
  - ต่อปั๊มลม + air stone ข้างใน (สำหรับ K1 Moving Bed)
  - ราคาถัง: 100-300 บาท

แบคทีเรีย (Probiotics):
  - ชนิด: Nitrosomonas + Nitrobacter
  - แบรนด์: Microbe-Lift, API, หรือท้องถิ่น
  - ราคา: 300-600 บาท/ขวด
  - วิธีใช้: เติมตามฉลาก (seed bacteria)

⚠️ Cycling สำคัญมาก (2-4 สัปดาห์):
  ก่อนใส่ปู ต้อง cycle ระบบให้แบคทีเรียพร้อม:
  1. เติมน้ำ + เกลือ + เปิดระบบ
  2. เติม ammonia source: หยิบอาหารปลาสัก 1 หยิบมือ ทิ้งให้เน่า
  3. เติม bacteria starter (Probiotics)
  4. วัดค่าทุกวัน:
     - Ammonia จะพุ่งขึ้นก่อน (สัปดาห์ 1)
     - Nitrite จะพุ่งตาม (สัปดาห์ 2)
     - ทั้งคู่จะค่อยๆ ลดลง (สัปดาห์ 3-4)
  5. เมื่อ Ammonia = 0 และ Nitrite = 0 → ปลอดภัยใส่ปู!

  วิธีเร่ง cycle:
  - ใช้ filter media จากตู้ปลาเก่า (มีแบคทีเรียพร้อม)
  - เพิ่มปั๊มลมให้แรง (แบคทีเรียต้องการ O2)
  - อุณหภูมิ 28-30°C (แบคทีเรียโตเร็ว)

⚠️ สำคัญ:
  - อย่าล้าง media ด้วยน้ำประปา (คลอรีนฆ่าแบคทีเรีย)
  - ใช้น้ำในระบบล้าง
  - อย่ารีบใส่ปูก่อน cycle เสร็จ — ปูจะตาย!
```

### E. UV Sterilizer (ฆ่าเชื้อโรค - Optional)

```
หน้าที่:
  ✅ ฆ่าแบคทีเรียโรค
  ✅ ฆ่าไวรัส
  ✅ ฆ่าพยาธิ
  ✅ ควบคุม algae (สาหร่าย)

ขนาด: 9-15W สำหรับ 1000L

ราคา: 1,500-3,000 บาท

ยี่ห้อ:
  - Jebao
  - TMC
  - หรือท้องถิ่น

วิธีติดตั้ง:
  1. ต่อหลัง biological filter
  2. ก่อน distribution manifold
  3. ตั้ง flow rate ตาม spec (ช้าๆ)

⚠️ หมายเหตุ:
  - Optional สำหรับ MVP
  - สามารถเพิ่มทีหลังได้
  - ถ้าปูแข็งแรงดี → อาจไม่จำเป็น
```

### F. Aeration System (ระบบให้ออกซิเจน)

```
หน้าที่:
  ✅ เพิ่มออกซิเจนในน้ำ
  ✅ ปูต้องการ DO > 5 ppm

Components:

1. Air Pump
   - ขนาด: 40-60W
   - กำลัง: 40-60 L/min
   - ราคา: 500-1,000 บาท

2. Air Tubing
   - ขนาด: 4-6 mm
   - ความยาว: 10-20 เมตร
   - ราคา: 100-200 บาท

3. Air Stones
   - จำนวน: 5-10 อัน
   - ขนาด: 5-10 cm
   - ราคา: 20-50 บาท/อัน

4. Check Valve (ป้องกันน้ำไหลย้อน)
   - จำนวน: 1-2 อัน
   - ราคา: 50 บาท/อัน

วิธีติดตั้ง:
  1. วาง air pump เหนือระดับน้ำ
  2. ต่อ check valve
  3. ต่อ air tubing
  4. ต่อไปที่ air stones
  5. วาง air stones ใน tank

ตำแหน่ง Air Stones:
  - ใน central tank: 3-5 อัน
  - ใน bio-filter: 1-2 อัน
  - ในกล่องปู: ไม่จำเป็น (น้ำไหลแล้ว)
```

### G. Individual Boxes (กล่องปู)

```
หน้าที่:
  ✅ ที่อยู่ปู (1 ตัว/กล่อง)
  ✅ ที่ซ่อนตอนลอกคราบ
  ✅ ป้องกันปูทำร้ายกัน

ขนาด: 30×40×15 cm
วัสดุ: HDPE plastic (ทนทาน)
จำนวน: 20 ใบ
ราคา: 40-50 บาท/ใบ

ภายในกล่อง:

1. Substrate (ทราย)
   - ชนิด: ทรายละเอียด 0.5-2 mm
   - ความลึก: 3-5 cm
   - ปริมาณ: 3-5 กก./กล่อง
   - เปลี่ยน: ทุก 2-4 สัปดาห์
   - ราคา: 300 บาท (20 กล่อง)

2. Inlet Pipe (ท่อน้ำเข้า)
   - ตำแหน่ง: มุมกล่อง
   - ขนาด: ¼" (6 mm)
   - สูง: เหนือทราย 5-10 cm

3. Outlet Pipe (ท่อน้ำออก) — แบบ Overflow
   - ตำแหน่ง: มุมตรงข้าม (ทแยงกับ inlet)
   - ขนาด: ¼" (6 mm)
   - สูง: ระดับน้ำต้องการ (10 cm)
   - หลักการ: น้ำไหลเข้า inlet → วิ่งผ่านกล่อง → ล้นออก outlet

⚠️ สำคัญมาก: น้ำต้องไหลผ่านตลอด!
   - น้ำนิ่ง = เน่าเร็ว = ปูตาย
   - Flow rate ที่แนะนำ: 1-2 ลิตร/นาที/กล่อง
   - คำนวณ: 20 กล่อง × 1.5 L/min = 30 L/min = 1,800 L/hr
   - ปั๊ม ½ HP (500-1000 L/hr) อาจไม่พอ!
     → ทางแก้: ลด flow rate เหลือ 0.5 L/min/กล่อง
     → หรือใช้ปั๊มที่แรงขึ้น (¾ HP)
   - วิธีเช็ค: เปิดปั๊ม → ดูน้ำไหลจาก outlet ทุกกล่อง
     ถ้ากล่องไหนน้ำไม่ไหล/ไหลน้อย → ปรับ valve

วิธีเจาะกล่อง HDPE:
  1. ใช้สว่านไฟฟ้า + ดอกสว่าน 8mm (สำหรับท่อ ¼")
  2. เจาะช้าๆ อย่ากดแรง (HDPE จะแตก)
  3. ขัดขอบรูด้วยกระดาษทราย 180
  4. ใส่ bulkhead fitting (ข้อต่อทะลุผนัง) + rubber washer
     - ค้น: "bulkhead fitting 1/4" หรือ "ข้อต่อทะลุถัง"
     - ราคา: 15-30 บาท/ตัว
  5. ขันให้แน่น + ทา silicone sealant รอบข้อต่อ
  6. รอ silicone แห้ง 24 ชม. ก่อนทดสอบ

  ⚠️ ห้ามใช้กาว PVC กับ HDPE (ไม่ติด!)
  ⚠️ ใช้ silicone aquarium grade เท่านั้น (ปลอดภัยกับสัตว์น้ำ)

วิธีติดตั้ง:
  1. เจาะรู 2 รู (inlet + outlet) ตามวิธีข้างบน
  2. ใส่ bulkhead fitting + seal
  3. ใส่ทราย
  4. วางเรียงแถว (4 แถว × 5 กล่อง)
  5. ต่อท่อน้ำ (hose ¼")
  6. ทดสอบด้วยน้ำจืดก่อน → เช็คทุกกล่องว่าน้ำไหลผ่าน
```

### H. Piping System (ระบบท่อ)

```

### ขั้นตอนที่ 7: ทดสอบระบบ (น้ำจืด)

```
Phase A: ทดสอบท่อและการไหล (1-2 วัน)

1. Fill tank ด้วยน้ำจืด (ไม่มีเกลือ)
2. เปิด pump
3. ตรวจสอบ:
   ☑️ ทุกกล่องมีน้ำไหลเข้าไหม?
   ☑️ ทุกกล่องน้ำไหลออกไหม?
   ☑️ น้ำไหลต่อเนื่อง ไม่ใช่แค่ขัง? (สำคัญ!)
   ☑️ มีท่อรั่วไหม?
   ☑️ อัตราการไหลสม่ำเสมอทุกกล่องไหม?
   ☑️ Pump ทำงานปกติไหม?
   ☑️ Protein Skimmer มีฟองไหม? (ยังไม่ต้องสมบูรณ์)

4. ปิด pump และตรวจสอบ:
   ☑️ น้ำไม่ไหลย้อนกลับไหม?
   ☑️ Check valve ทำงานไหม?

5. แก้ไขปัญหาที่พบ
6. ทดสอบซ้ำ 24 ชม.
7. ถ้าทุกอย่าง OK → เตรียมน้ำเกลือ
```

### ขั้นตอนที่ 8: Cycling ระบบ (2-4 สัปดาห์ — ห้ามข้าม!)

```
Phase B: สร้างแบคทีเรียในระบบ

วันที่ 1:
  1. เปลี่ยนน้ำจืด → น้ำเกลือ (15-20 ppt)
  2. เปิดระบบทั้งหมด (pump + air + skimmer)
  3. เติม ammonia source: อาหารปลาสัก 1 หยิบมือ ทิ้งให้เน่า
  4. เติม bacteria starter (probiotics)

ทุกวัน:
  ☑️ วัด Ammonia (NH3)
  ☑️ วัด Nitrite (NO2)
  ☑️ วัด pH + Temperature
  ☑️ บันทึกค่าใน log

จะเห็นแบบนี้:
  สัปดาห์ 1: Ammonia พุ่งขึ้น ↑↑ (ปกติ)
  สัปดาห์ 2: Ammonia ลด ↓ + Nitrite พุ่ง ↑↑ (แบคทีเรียเริ่มทำงาน)
  สัปดาห์ 3: Nitrite ลด ↓↓ (เกือบเสร็จ)
  สัปดาห์ 4: Ammonia = 0, Nitrite = 0 ✅ พร้อมใส่ปู!

วิธีเร่ง cycle ให้เร็วขึ้น:
  - ใช้ filter media จากตู้ปลาเก่า
  - เปิดปั๊มลมแรงๆ (O2 ช่วยแบคทีเรียโต)
  - อุณหภูมิ 28-30°C
  - ห้ามเปลี่ยนน้ำระหว่าง cycle (ยกเว้น pH ต่ำมาก)

⚠️ สำคัญ:
  - อย่ารีบใส่ปูก่อน Ammonia=0 และ Nitrite=0
  - ถ้า cycle ไม่เสร็จแล้วใส่ปู → ปูตายแน่!
  - อดทนรอ 2-4 สัปดาห์ = ประกันชีวิตปู
```

   ✅ บันทึกค่าน้ำทุกวัน (pH, salinity, temp)

```

### ความผิดพลาดที่พบบ่อย

```
❌ ท่อรั่ว
   แก้: ใช้ PVC glue ให้มากพอ, รอให้แห้งสนิท

❌ น้ำไหลไม่สม่ำเสมอ
   แก้: ใส่ valve ควบคุมแต่ละแถว

❌ น้ำไหลย้อนกลับ
   แก้: ใส่ check valve

❌ Pump ดังผิดปกติ
   แก้: ตรวจสอบ air lock, prime pump ใหม่

❌ Filter อุดตัน
   แก้: ล้าง mechanical filter สัปดาห์ละครั้ง

❌ แบคทีเรียตาย
   แก้: อย่าล้าง bio-filter ด้วยน้ำประปา

❌ ปูตาย
   แก้: ตรวจสอบคุณภาพน้ำ, acclimation ให้ถูกวิธี
```

### ระบบสำรอง (Backup — จำเป็นมาก!)

```
⚠️ ถ้าปั๊มลมพัง → ปูตายภายไม่กี่ชั่วโมง!
⚠️ ถ้าไฟดับ → ปูตายถ้าไม่มี backup!

1. ปั๊มลมสำรอง (จำเป็น!)
   - ซื้อปั๊มลมตัวเล็กเก็บไว้ 1 ตัว
   - ราคา: 300-500 บาท
   - ค้น: "ปั๊มลม aquarium สำรอง"
   - เก็บไว้ใกล้มือ พร้อมใช้ทันที

2. UPS / Battery Backup (แนะนำอย่างยิ่ง)
   - ปั๊มลม DC + แบตเตอรี่ (ทำงานได้แม้ไฟดับ)
   - ราคา: 500-1,500 บาท
   - ค้น: "ปั๊มลม DC แบตเตอรี่", "air pump battery backup"
   - แบต: ทำงานได้ 6-12 ชม. ขึ้นกับขนาด
   - ⭐ ตัวที่ดี: สลับอัตโนมัติเมื่อไฟดับ (auto switch)

3. สรุปของสำรอง:
   □ ปั๊มลมสำรอง AC (300-500 บาท) — จำเป็น
   □ ปั๊มลม DC + แบต (500-1,500 บาท) — แนะนำ
   □ ท่อ/ข้อต่อสำรอง (เก็บไว้ 2-3 ชิ้น) — nice to have
```

---

### การแก้ปัญหาฉุกเฉิน

```
⚠️ ไฟดับ
  → ปั๊มลม DC + แบตเตอรี่ ทำงานอัตโนมัติ
  → ถ้าไม่มี → ใช้ปั๊มลมสำรอง AC + ไฟจากรถ/เพื่อนบ้าน
  → หยุดให้อาหาร (ลดการใช้ O2)
  → ถ้าดับนาน > 4 ชม. → เปลี่ยนน้ำบางส่วน

⚠️ ปั๊มลมพัง
  → สลับปั๊มลมสำรองทันที!
  → สั่งปั๊มใหม่ในวันเดียวกัน
  → ถ้าไม่มีสำรอง → ตีน้ำด้วยมือ + ซื้อใหม่ด่วน

⚠️ Pump น้ำพัง
  → หยุดให้อาหารทันที
  → เปิดปั๊มลมแรงขึ้น (ชดเชย O2)
  → ซื้อ pump ใหม่เร็วที่สุด

⚠️ น้ำรั่วหนัก
  → ปิด pump ทันที
  → หาจุดรั่ว
  → ซ่อม/เปลี่ยนท่อ

⚠️ ปูตายหลายตัว
  → ตรวจสอบคุณภาพน้ำทันที
  → pH, salinity, temperature
  → เปลี่ยนน้ำ 50% ถ้าจำเป็น
  → แยกปูตัวที่แข็งแรงออก

⚠️ โรคระบาด
  → แยกปูตัวที่ป่วย
  → ลดความหนาแน่น
  → เพิ่มการกรอง
  → ปรึกษาผู้เชี่ยวชาญ
```
